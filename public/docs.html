CSS Framework

The basic framework provides a tidy typographic style and the following utility CSS classes:

container
All content should sit inside a container. It keeps content to a sensible width, off the edges of the browser window and centers itself on the page.
textcenter
Used to align text to the centerline
textright
Used to align text to the right edge.
textleft
Used to align text to the left edge.
aligncenter
Centers a block level element inside it's parent.
alignright
Aligns a block level element to the right of its parent.
alignleft
Aligns a block level element to the left of its parent.
clearfix
When elements are floated, they lose explicit height and the parent can become zero height. Applying this class to the parent fixes that.

Backend API

Ensure all requests are sent with the header Content-Type: application/json or the request will fail to log.

You can test your integration by monitoring the logs in the storage folder

The backend JSON API provides the following endpoints:

Analytics endpoint: http://localhost:8080/analytics

Method:
POST
Required input format:
{"url":"http://localhost:8080/challenge.html"}

Newsletter endpoint: http://localhost:8080/newsletter

Method:
POST
Required input format:
{"name": "users name here", "email": "users-email@address.com"}
